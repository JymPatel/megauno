## Automatic Door Light
it's a project by which LED turns on if Ultrasonic Sensor detects distance more than 10 cm.

#### Requienments
arduino board
ultrasonic sensor (HC-SR04)
led
jumper wires

#### How to run
put code in your arduino IDE.
compile, upload, enjoy!
